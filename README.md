# 🧮 Calculadora Básica

## 📌 Descripción
La calculadora es una herramienta diseñada para realizar operaciones matemáticas básicas de manera rápida y sencilla. Permite efectuar sumas, restas, multiplicaciones y divisiones introduciendo los valores numéricos correspondientes.

Su funcionamiento es intuitivo, lo que facilita su uso tanto para cálculos cotidianos como para tareas más específicas, proporcionando resultados precisos de forma inmediata.

---

## ⚙️ Funcionalidades

- ➕ Suma de números
- ➖ Resta de números
- ✖️ Multiplicación de números
- ➗ División de números
- ✔️ Resultados instantáneos
- 🖥️ Interfaz sencilla e intuitiva

---

## 🚀 Uso

1. Introduce el primer número.
2. Selecciona la operación deseada.
3. Introduce el segundo número.
4. Obtén el resultado automáticamente.
